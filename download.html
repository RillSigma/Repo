Unduh DarkPS

Pilih versi yang sesuai dengan perangkat Anda untuk memulai petualangan.

Windows Android iOS

Cara Unduh di Windows

  1. Tekan ⊞ Win + R untuk membuka Run
  2. Ketik C:\Windows\System32\drivers\etc lalu Enter
  3. Klik kanan file hosts → buka dengan Notepad (Admin)
  4. Tambahkan host dari tombol Copy IP
  5. Simpan file, buka Growtopia!

Cara Unduh di Android

  1. Unduh PowerTunnel
  2. Klik tombol PowerTunnel di atas
  3. Buka PowerTunnel → Host Settings
  4. Tempel URL, ubah host update period ke "on start"
  5. Buka Growtopia dan nikmati servernya!

Cara Unduh di iOS

  1. Unduh Surge 5
  2. Masuk ke menu IMPORT → Download Profile from URL
  3. Tempel URL dari tombol di atas
  4. Setup dan izinkan konfigurasi VPN
  5. Buka Growtopia dan mainkan!
